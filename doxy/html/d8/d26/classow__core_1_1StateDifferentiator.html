Detailed Description

template<typename _Derived>
class ow_core::StateDifferentiator< _Derived >

The StateDifferentiator class.

This class takes the current position of a state class and updates the current velocity and accelerations.

The _Derived template parameter has to be derived from ow_core::StateBase thus belong to the group of state type classes.
