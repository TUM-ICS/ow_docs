Create the typedefs for vectors and matrices.

Create the typedefs type Type and append the type suffix TypeSuffix to the name.

E.g. OPEN_WALKER_CORE_TYPEDEFS_ALL_SIZES(unsigned int,ui) will create vectors and matrices such as Vector1ui, Matrix3ui, etc.
