Detailed Description

The IHwInterface class.

This is the Interface class that abstracts the robot and gives access to its jointstate and sensor readings.

IHwInterface is used by the controller_base class.

This class is special since it directly couples to the ros_control::interface in the class ros_control_plugin.h
