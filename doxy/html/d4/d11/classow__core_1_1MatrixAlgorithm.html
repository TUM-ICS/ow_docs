Detailed Description

template<typename _Derived>
class ow_core::MatrixAlgorithm< _Derived >

The MatrixAlgorithm class.

This class provieds the base to apply Algorithms for all vectors and matrices by applying them elementwise. The _Derived template parameter has to be derived from Eigen::Matrix.
