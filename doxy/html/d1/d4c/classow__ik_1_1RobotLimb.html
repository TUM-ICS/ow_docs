Detailed Description

The RobotLimb class.

This class calculates the kinematics and dynamics of a RobotLimb. It takes as an input a reference to a sub-vector of the complete body state and modifies this vector upon function calls.

Mutliple Robotlimbs presend in the robot_model.h to build the complete robot.
