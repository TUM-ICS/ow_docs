Detailed Description

template<typename _Scalar>
class ow_core::ScalarButterWorthFilter< _Scalar >

The ScalarButterWorthFilter class.

ScalarButterWorthFilter is a lowpass filter for scalar values with variable filter ORDER. Implementation is based on: http://www.exstrom.com/journal/sigproc/

Constructor a filter from a second-order sections form. A Matrix containing the coefficients of multiple second-order filters. Matrix dimension must be Nx6, with N the numer of second-order systems and 6 the 3 nominator coefficients b + 3 denominator coefficients a.

See matlab function: sos = ss2sos(...)

Parameters
A_sosMatrix of second-order sections
