Detailed Description

The HomeingController class.

This Class uses some modules of the framework to execute the homing motion of the robot. It obtains the real robot data through the ow::IHwInterface and calls the update functions on each module. Finally the HomeingController produces a new joint command that is then send to to the robot.

This controller can be enabled in the ow_ros_control_plugin package. It is mainly used for debugging.
