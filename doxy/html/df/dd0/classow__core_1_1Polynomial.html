Detailed Description

template<typename _Scalar>
class ow_core::Polynomial< _Scalar >

The Polynomial class.

Takes the coefficients of a polynomial and offers function to evaluate it. This class operates with scalar types only. For multiple dimensions use polynomial_trajectory.

Polynomial coeffs are in the form: a0 + a1*x + a2*x^2 + ... and stored as coeffs = [a0, a1, a2, ...]

Polynomials start at time_start = 0 and end at time period.
