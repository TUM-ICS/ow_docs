Detailed Description

template<typename _Scalar>
class ow_core::ScalarFiniteDifference< _Scalar >

The static ScalarFiniteDifference class.

Additional information on the implemented algorithm:

  • https://en.wikipedia.org/wiki/Finite_difference_coefficient
  • http://web.media.mit.edu/~crtaylor/calculator.html

Implements a finite difference numeric derivative using the backward form.

The order of the derivatrive , and the stencil can be specified.

Since we implement a backward differentiation the stencil must have only integer coefficients .

The order of accuracy is defined by the stencil length . The longer the stencil, the higher the accuracy. The stencil length has to be greather than the order of the derivative : .
