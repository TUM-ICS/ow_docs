Detailed Description

template<typename _Scalar>
class ow_core::ScalarExponentialDecayFilter< _Scalar >

The ScalarExponentialDecayFilter class.

ScalarExponentialDecayFilter is a filter for smoothing of time series data using the exponential window function.

See: https://en.wikipedia.org/wiki/Exponential_smoothing
